Empezamos las pruebas con github

1) Descargue git y la UI de github
2) Hice un fork del proyecto Prueba
3) Clone el proyecto Prueba a mi repo local
4) Hice un txt de prueba con los pasos que estoy haciendo
5) Hago un commit a mi repo local (para agregar este txt)
6) Hago un push para subir los cambios a mi repo de github usando el comando "git push origin master" donde origin seria mi repo de github huberalberti/prueba y master la rama (branch)
7) Solicito el pull request para que los cambios de mi repo se apliquen al init-swf/prueba
8) Fin del demo
